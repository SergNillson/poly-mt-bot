# 🤖 Polymarket Copy Trading Bot

Автоматический бот для копирования сделок успешных трейдеров на Polymarket с режимом paper trading и уведомлениями в Telegram.

## 🎯 Что это?

Этот бот отслеживает сделки выбранного трейдера на платформе Polymarket и автоматически копирует их в режиме paper trading (виртуальная торговля без реальных денег).

**Основная идея:** Вы следите за успешным трейдером, бот копирует его сделки пропорционально вашему балансу, и вы можете отслеживать результаты через Telegram.

## ✨ Основные возможности

- ✅ **Paper Trading режим** - тестирование стратегии без риска
- ✅ **Отслеживание сделок** - мониторинг транзакций целевого трейдера
- ✅ **Пропорциональное копирование** - масштабирование позиций под ваш баланс
- ✅ **Telegram уведомления** - получайте информацию о новых сделках и позициях
- ✅ **Telegram команды** - управление и просмотр статистики через бота
- ✅ **SQLite база данных** - хранение истории сделок и позиций
- ✅ **Подробное логирование** - полная прозрачность работы

## 🚀 Быстрый старт

### Шаг 1: Клонирование репозитория
```bash
git clone https://github.com/SergNillson/pm-copy-bot.git
cd pm-copy-bot
```

### Шаг 2: Создание виртуального окружения
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Mac/Linux
source venv/bin/activate
```

### Шаг 3: Установка зависимостей
```bash
pip install -r requirements.txt
```

### Шаг 4: Настройка конфигурации

Создайте файл `.env` в корне проекта:
```bash
cp config/.env.example .env
```

Отредактируйте `.env` файл:
```env
# Адрес трейдера для отслеживания
TRACKED_TRADER_ADDRESS=0x63ce342161250d705dc0b16df89036c8e5f9ba9a

# Telegram настройки
TELEGRAM_BOT_TOKEN=your_bot_token_here
TELEGRAM_CHAT_ID=your_chat_id_here

# Параметры копирования
COPY_RATIO=1.0              # 1.0 = 100% копирование размера
MAX_POSITION_SIZE=100.0     # Максимальный размер позиции в USD
CHECK_INTERVAL=60           # Интервал проверки в секундах
```

#### Как получить Telegram Bot Token и Chat ID:

**Bot Token:**
1. Найдите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Скопируйте полученный токен

**Chat ID:**
1. Найдите [@userinfobot](https://t.me/userinfobot) в Telegram
2. Отправьте `/start`
3. Скопируйте ваш ID

### Шаг 5: Запуск бота
```bash
python main.py
```

## 📱 Telegram команды

После запуска бота вы можете использовать команды:

- `/start` - Начало работы, показать список команд
- `/stats` - Показать статистику торговли (P&L, винрейт, позиции)
- `/positions` - Показать открытые позиции с текущими ценами
- `/help` - Справка по командам

## ⚙️ Конфигурация

### Основные параметры (.env файл):

| Параметр | Описание | Пример |
|----------|----------|--------|
| `TRACKED_TRADER_ADDRESS` | Адрес кошелька трейдера для копирования | `0x63ce...` |
| `TELEGRAM_BOT_TOKEN` | Токен Telegram бота | `123456:ABC...` |
| `TELEGRAM_CHAT_ID` | ID чата для уведомлений | `987654321` |
| `COPY_RATIO` | Коэффициент копирования (0.1 = 10%, 1.0 = 100%) | `1.0` |
| `MAX_POSITION_SIZE` | Максимальный размер позиции в USD | `100.0` |
| `CHECK_INTERVAL` | Интервал проверки новых сделок (секунды) | `60` |

## 🔍 Как это работает

1. **Мониторинг**: Бот периодически проверяет новые сделки целевого трейдера через Polymarket API
2. **Уведомление**: Отправляет уведомление в Telegram о новой сделке
3. **Копирование**: Открывает виртуальную позицию с учетом коэффициента копирования
4. **Отслеживание**: Следит за текущими ценами и рассчитывает нереализованный P&L
5. **Статистика**: Показывает результаты через команды Telegram бота

## 📝 База данных

Результаты сохраняются в `data/trading.db` (SQLite).

**Таблицы:**
- `tracked_trades` - все отслеженные сделки трейдера
- `positions` - ваши виртуальные позиции (открытые и закрытые)

Для просмотра БД можно использовать:
- [DB Browser for SQLite](https://sqlitebrowser.org/) (GUI)
- Командная строка: `sqlite3 data/trading.db`

## ⚠️ Важно

**Это paper trading версия!** Никакие реальные сделки не совершаются. Бот только симулирует торговлю.

Для реальной торговли потребуется:
- Реальный аккаунт Polymarket
- Подключение кошелька
- API ключи
- Дополнительная доработка кода

## 🛠 Troubleshooting

**Бот не запускается:**
- Проверьте Python версию: `python --version` (нужен 3.8+)
- Активируйте виртуальное окружение
- Переустановите зависимости: `pip install -r requirements.txt --upgrade`
- Проверьте `.env` файл

**Не приходят уведомления в Telegram:**
- Проверьте правильность `TELEGRAM_BOT_TOKEN`
- Убедитесь, что написали боту `/start`
- Проверьте `TELEGRAM_CHAT_ID`

**Нет новых сделок:**
- Проверьте адрес трейдера в `.env`
- Убедитесь, что трейдер активен
- Посмотрите логи: `bot.log`

**Ошибки API:**
- Polymarket API иногда недоступен, бот будет повторять запросы
- Проверьте интернет-соединение
